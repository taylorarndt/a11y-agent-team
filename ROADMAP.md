# Roadmap

Track individual items on the [issues board](https://github.com/Community-Access/accessibility-agents/issues?q=label%3Aroadmap) where you can comment, subscribe, and follow progress.

## Planned

### High Priority

The following items are the highest-priority features planned for the next release.

| Feature | Description | Issue |
|---------|-------------|-------|
| Mobile native accessibility agents | iOS (UIAccessibility/SwiftUI) and Android (TalkBack) agents for native app accessibility enforcement | [#8](https://github.com/Community-Access/accessibility-agents/issues/8) |
| Anthropic Connectors Directory listing | Publish to the Anthropic Connectors Directory so Claude Desktop users get automatic installs and updates | [#9](https://github.com/Community-Access/accessibility-agents/issues/9) |

### Medium Priority

The following items are planned after high-priority work is complete.

| Feature | Description | Issue |
|---------|-------------|-------|
| veraPDF integration | Full PDF/UA validation via veraPDF CLI for comprehensive, standards-grade PDF checking | [#10](https://github.com/Community-Access/accessibility-agents/issues/10) |
| Document remediation tools | MCP tools that fix accessibility issues in Office and PDF documents, not just detect them | [#11](https://github.com/Community-Access/accessibility-agents/issues/11) |

### Low Priority

The following items are planned but not yet scheduled.

| Feature | Description | Issue |
|---------|-------------|-------|
| WCAG AAA agent | Dedicated agent for AAA-level conformance checking beyond the current AA target | [#12](https://github.com/Community-Access/accessibility-agents/issues/12) |
| Multi-language support | Agent instructions and documentation in languages beyond English | [#14](https://github.com/Community-Access/accessibility-agents/issues/14) |

## Completed

See the [changelog](https://github.com/Community-Access/accessibility-agents/releases) for shipped features.

| Feature | Description |
|---------|-------------|
| Markdown Accessibility Agents | `markdown-a11y-assistant` orchestrator with parallel `markdown-scanner` and `markdown-fixer` sub-agents. Covers 9 domains: links, alt text, headings, tables, emoji, diagrams, em-dashes, anchor links, and plain language. Includes 4 custom prompts, a `markdown-accessibility` skill, and full Copilot/Claude Code parity. |
